# Sobre o Projeto

Este projeto não tem o objetivo de fazer um fluxo de loja -apesar de ter a página incial com produtos-, mas criar a funcionalidade de carrinho usando o UseReducer, do ReactJs, e implementar consumo de API.

## Tecnologias Usadas
- NextJs 16.1.6
- ReactJs 19.2.3
- Typescript ^5
- TailwindCSS ^4 + PostCSS
- Axios ^1.13.4
- Eslint ^9
